# LinkedIn Ad Blocker

A simple Chrome extension that hides "Promoted" posts on LinkedIn by removing sponsored content from the feed.

## Features
- Removes LinkedIn "Promoted" posts
- Runs automatically on page load
- Lightweight and fast

## Installation (Developer Mode)
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions`.
3. Enable **Developer Mode**.
4. Click **Load unpacked**.
5. Select the project folder.

## Files
- `manifest.json` — Extension configuration
- `background.js` — Listens for LinkedIn page loads
- `linkedin.js` — Removes promoted posts from the feed
